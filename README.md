1. If you don't have brew

```
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)" ; echo "export PATH=/opt/homebrew/bin:$PATH" >> ~/.zshrc && brew install go
```


else:

```
brew install go
```


see go working:
```
go version
```
should print something.
